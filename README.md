# 计量设备待装超时提醒与自动化统计系统

本项目用于自动化读取、过滤、去重与汇总计量设备（电能表、互感器、负荷管理终端、智能融合终端等）的待装明细数据，精确计算扣除节假日/调休后的工作日超时天数，并生成通报文案及无宏 Excel 统计报表。

---

## 核心功能特性

1. **精准工作日计算 (2026年)**
   - 自动扣除 2026 年国家法定节假日（元旦、春节、清明、劳动、端午、中秋、国庆等）。
   - 自动识别 2026 年调休工作日。
   - 精确计算设备从“最后一次状态改变时间”到当前统计时间的工作日时长。

2. **多源 Excel 明细数据自动合并**
   - 自动匹配指定日期目录中的原始设备明细文件（`电能表*.xlsx`、`互感器*.xlsx`、`负荷管理终端*.xlsx` 等）。
   - 智能识别表头位置，提取资产编号、设备类别、县区局、供电所及改变时间。
   - 自动计算并标记：**超 3 个工作日**、**超 10 个工作日**、**超 15 个工作日**及**上月领出**。

3. **区局数据分析与通报文案自动生成**
   - 自动规范化并简化区局名称（如：“城区局”、“东南局”、“虎门局”等）。
   - 按照“超 3 个工作日”台数对区局降序排列，对存在超期设备的数据施加黄色预警填充。
   - 自动生成标准待装超时监控通报文本，直接写入 `区局分析` 工作表。

4. **供电服务中心直属设备统计**
   - 筛选无县区局归属的直属/服务中心设备。
   - 按设备类别（电能表、互感器、负荷管理终端、其他设备）汇总统计，并自动应用优雅的格式与斑马纹变色样式。

5. **无宏 Excel 导出与系统剪贴板自动复制**
   - 导出为标准的无 VBA 宏纯 `.xlsx` 格式（如 `待装提醒20260718.xlsx`）。
   - 基于 Windows 原生 API (`CF_HDROP` 剪贴板格式)，运行成功后**自动将生成的 Excel 报表文件直接复制到系统剪贴板**，支持在微信、钉钉或邮件中通过 `Ctrl + V` 直接粘贴发送文件。

## 使用说明

### 方式一：使用 `.exe` 程序（推荐无 Python 环境用户）

1. 在当前目录下创建当天的日期文件夹（格式为 8 位数字 `YYYYMMDD`，例如 `20260718`）。
2. 将 `待装提醒模板.xlsm` 以及待处理的设备明细 Excel 文件（`.xlsx`）放入该日期文件夹中。
3. 双击运行 `process_reminders.exe`。
4. 处理完成后，文件夹内将生成 `待装提醒YYYYMMDD.xlsx` 统计报表。
5. 此时报表文件已自动复制到系统剪贴板，您可以直接在聊天软件或邮件附件中按下 `Ctrl + V` 粘贴发送该文件。

### 方式二：通过 Python 源码运行

1. **安装依赖**
   ```bash
   pip install openpyxl
   ```

2. **准备数据**
   确保当天的 `YYYYMMDD` 子目录中存在 `待装提醒模板*.xlsm` 和待合并的 `*.xlsx` 数据文件。

3. **运行脚本**
   ```bash
   python process_reminders.py
   ```
